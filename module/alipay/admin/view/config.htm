支付宝设置

该企业支付宝账号对应的合作身份者id，以2088开头的16位纯数字
{echo Template::radio('alipay[transport]', array('http' => 'http', 'https' => 'https'), $alipay['transport'])} 若能通过https访问支付宝，请选择https，否则选择http
{echo Template::radio('alipay[enable_single_trade_query]', array('否', '是'), !empty($alipay['enable_single_trade_query']))} 请先开通支付宝单次交易查询接口
设置此地址后默认URL失效，仅通知该自定义URL
./data/alipay_private_key_.pem 签名方式设置为“RSA”时必填
./data/alipay_public_key_.pem 签名方式设置为“RSA”时必填
